== Description ==

Who links to you? What says about you? iMasters WP Incoming links track, in real time, all sources that is generating traffic to your WordPress based site.

== Installation ==

How to install iMasters WP Incoming Links

e.g.

1. Upload `imasters-wp-incoming-links` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Incoming Links' menu in WordPress
